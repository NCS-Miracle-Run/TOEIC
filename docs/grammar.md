# GRAMMAR

## 단수 취급하는 표현 정리

### 1. 부정대명사 (항상 단수!)

| 표현 | 뜻 |
|------|----|
| anyone / anybody | 누구든지 |
| everyone / everybody | 모든 사람 |
| someone / somebody | 누군가 |
| no one / nobody | 아무도 |
| each | 각각 |
| either | 둘 중 하나 |
| neither | 둘 다 아닌 |

> Everyone **is** ready. (O) / Everyone **are** ready. (X)

### 2. each / every + 명사

`each` / `every` 뒤에 오는 명사는 무조건 단수

- Each employee **has** a locker.
- Every department **submits** a report.

### 3. one of + 복수명사

동사는 `one`에 맞춰 단수

- One of the managers **is** absent today.

## -ed vs -ing 형용사 구분

### 핵심 판단 기준

| 형태 | 의미 | 주어/수식 대상의 역할 |
|------|------|--------------------|
| **-ing** | 감정·상태를 **유발하는** 쪽 | 원인, 사물, 상황 |
| **-ed** | 감정·상태를 **느끼는** 쪽 | 결과, 사람 |

> **한 줄 요약**: 사람이 느끼면 -ed / 상황·사물이 유발하면 -ing

---

### 감정 형용사 쌍 (토익 빈출)

| -ing (유발) | -ed (느낌) | 뜻 |
|------------|-----------|-----|
| interesting | interested | 흥미로운 / 흥미를 느끼는 |
| boring | bored | 지루한 / 지루함을 느끼는 |
| exciting | excited | 신나는 / 신남을 느끼는 |
| surprising | surprised | 놀라운 / 놀람을 느끼는 |
| satisfying | satisfied | 만족스러운 / 만족을 느끼는 |
| confusing | confused | 혼란스러운 / 혼란을 느끼는 |
| disappointing | disappointed | 실망스러운 / 실망을 느끼는 |
| exhausting | exhausted | 기진맥진하게 하는 / 기진맥진한 |
| encouraging | encouraged | 고무적인 / 용기를 얻은 |
| overwhelming | overwhelmed | 압도적인 / 압도당한 |

**예문 비교**
- The presentation was **interesting**. (발표 자체가 흥미를 유발)
- I was **interested** in the presentation. (내가 흥미를 느낌)
- The result was **disappointing**. (결과가 실망감을 유발)
- She was **disappointed** with the result. (그녀가 실망을 느낌)

### 분사가 명사를 수식할 때

감정 형용사가 아니라 명사를 꾸미는 분사일 때는 **능동/수동** 관계로 판단.

| 형태 | 관계 | 예문 |
|------|------|------|
| **-ing** | 명사가 동작을 **하는** 쪽 (능동) | the **remaining** items (남아 있는 항목들) |
| **-ed** | 명사가 동작을 **당하는** 쪽 (수동) | the **attached** file (첨부된 파일) |

**토익 빈출 표현**

| 표현 | 뜻 |
|------|----|
| the **attached** file | 첨부된 파일 |
| the **updated** schedule | 업데이트된 일정 |
| the **revised** plan | 수정된 계획 |
| the **expected** results | 예상된 결과 |
| the **following** items | 다음 항목들 |
| the **remaining** seats | 남은 좌석들 |
| the **increasing** demand | 증가하는 수요 |
| a **challenging** task | 도전적인 과제 |
