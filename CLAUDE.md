# TOEIC 스터디 프로젝트

## 프로젝트 목적
- 토익 공부 기록 관리
- 클로드에게 토익 문제를 받아서 풀이 연습

---

## 문제 출제 요청 방법

문제를 받고 싶을 때는 다음과 같이 요청하면 됩니다:

- `파트 5 문제 5개 줘`
- `LC 파트 3 유형으로 스크립트 + 문제 만들어줘`
- `어휘 문제 10개 줘`
- `틀린 문제 복습 문제 만들어줘`

---

## 출제 규칙 (클로드가 따를 것)

1. **실제 토익 형식**을 최대한 따를 것
2. 문제마다 **정답 + 해설**을 함께 제공할 것
3. 틀린 문제가 있으면 **왜 틀렸는지** 명확히 설명할 것
4. 어휘/문법 문제는 **보기 4개** (A/B/C/D) 형식으로 출제할 것
5. 난이도는 별도 요청이 없으면 **중급(600~800점대)** 기준으로 출제할 것

---

## 파트별 출제 형식

### Part 5 (단문 빈칸 채우기)
- 문장 1개 + 보기 4개
- 정답 및 해설 포함

### Part 6 (장문 빈칸 채우기)
- 지문 1개 + 빈칸 4개
- 각 빈칸마다 보기 4개

### Part 7 (독해)
- 지문(단일/이중/삼중) + 질문 2~5개
- 정답 및 해설 포함

### LC (파트 1~4)
- 스크립트 형태로 제공
- 문제 + 보기 4개 + 정답 + 해설 포함
